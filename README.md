# BankProject

## Описание:

BankProject - это приложение на Python, разрабатываемое в рамках прохождения курса.
### В коде приложения реализован модуль "generators", содержащий в себе функции-генераторы:
- filter_by_currency - возвращает из списка транзакций транзакцию с заданной ввалютой;
- transaction_descriptions - возвращает описание транзакции;
- card_number_generator - Возвращает номера карт в определенном диапазоне в формате XXXX XXXX XXXX XXXX.

### В коде приложения реализован модуль "decorators", содержащий в себе декоратор log, отображающий в консоли либо в файле логирования:
- Имя функции;
- Время запуска и окончания функции;
- Статус функции
- Тип ошибки и введенные данные (при ошибке в исполнении функции)
## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/AranTvin/BankProject
```
2. Установите зависимости:
```
poetry install
```
