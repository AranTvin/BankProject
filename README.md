# Проект X

## Описание:

BankProject - это приложение на Python, разрабатываемое в рамках прохождения курса.

## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/AranTvin/BankProject
```
2. Установите зависимости:
```
poetry install
```
